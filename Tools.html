TOOLS HARDWARE

1. LINE FOLLOWER SENSOR : Line sensors detect the presence of a black line by emitting infrared (IR) light and detecting the light levels that return to the sensor.
2. VOICE RECOGNITION MODULE : Voice Recognition Module is a compact and easy-control speaking recognition board.
3. ARDUINO MEGA : The Arduino Mega 2560 is a microcontroller board based on the ATmega2560. It has 54 digital input/output pins (of which 15 can be used as PWM outputs), 16 analog inputs, 4 UARTs (hardware serial ports), a 16 MHz crystal oscillator, a USB connection, a power jack, an ICSP header, and a reset button.
